## Maintenance Guidelines

### Adding New Documentation
1. Identify the appropriate category (01-10)
2. Place file in corresponding folder
3. Update `Documentation/README.md` index if significant
4. Follow naming convention: `SCREAMING_SNAKE_CASE.md`

### Archiving Old Documents
1. Move completed/superseded docs to `10-Legacy/`
2. Add date prefix if needed: `YYYY_MM_DocumentName.md`
3. Update references in master index

### Folder Naming Convention
- Use numbered prefixes: `01-`, `02-`, etc.
- Use descriptive names with hyphens: `Getting-Started`, `AI-ML`
- Avoid spaces in folder names
